## How to Use This Package

1. **Start with the prospect list** — verify Square POS usage for top 10 before reaching out (look for Square links in their Instagram bio or website ordering button)
2. **Choose the right template** by channel — email for direct contacts, IG DM for social-active shops, LinkedIn for owner-identified contacts
3. **Use the one-liner** in any in-person or casual conversation
4. **Refer to the FAQ** when prospects push back or ask common questions
